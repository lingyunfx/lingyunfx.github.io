Maya Adv插件使用流程

zhangly 2021-06-20 09:05:11
Categories： DCC > Maya > 绑定学习笔记 Tags：

1.检查模型

  • 身体模型的中线，是否在网格中线（前视图检查）
  • 头部模型的中线，大致在网格中线附近（右视图检查）
  • 脚最底的面是否贴近地面（右视图检查）
  • 膝盖朝向正前方，手肘要朝向正后方（透视图检查）
  • 除大拇指的四根手指，在一个水平面（从正视图看）
  • 模型的正面是否朝向Z轴

2.整理模型

  • 层级结构：一般头部一个Group，身体一个Group。
  • 删除未使用的材质球。
  • 选择所有模型：居中，冻结，删除历史。

3.导入骨骼模板，调整根骨骼


选择大环缩放骨骼，将根骨骼的位置缩放到臀部中心。


调整骨骼显示大小，开启FitMode，实时更新骨骼朝向（x轴永远朝向子骨骼）


切换到右视图，将根骨骼调整到臀部中心。

然后切换到前视图，进行微调。

4.调整腿部骨骼

  • 大腿骨骼：位移XYZ，旋转YZ
  • 小腿骨骼：位移XY，旋转Z
  • 脚踝骨骼：位移XY，旋转X
  • 脚后跟骨骼：位移YZ，不旋转
  • 脚趾和脚尖骨骼：位移XY，不旋转
  • 脚掌两侧骨骼：位移XYZ，不旋转

切换到前视图


切换到右视图，移动到大腿中心


切换到前视图，旋转Y轴，将小腿骨骼和脚踝骨骼移动模型中间。（如果不能两者，优先脚踝骨骼在中间）


切换到右视图，调整小腿骨骼（膝盖位置）

**注意：**调整小腿和脚踝骨骼的时候，不能移动Z轴。


继续在右视图调整脚踝骨骼


切换到透视图，旋转脚掌的朝向。


注意：如果在旋转脚踝骨骼后，还需要继续改变脚踝骨骼的位置。

需要将被冻结转移的数值归零，再进行位置调整。


这么做的目的是为了让脚踝骨骼Y轴朝向正前方。


最后就是脚部骨骼调整

时刻检查极向量是否朝着正前方

5.胸部骨骼调整

胸部骨骼放在胸上，腹部骨骼需要在root骨骼和胸部骨骼中间部分。

6.手臂骨骼调整

  • 大臂骨骼：位移XYZ，旋转YZ
  • 小臂骨骼：位移XY，旋转Z
  • 手腕骨骼：位移XY，旋转XYZ
  • 手指根骨骼：没有限制
  • 手指子骨骼：位移XZ，旋转Y
  • 手指末端骨骼：位移X

调整肩部骨骼（前视图）

调整完位置后，旋转与肩部平行。


调整大臂骨骼位置后，旋转它的Y轴，使小臂骨骼和手腕骨骼居中（这个时候可以一边移动手腕的骨骼的位置到手腕，一边旋转大臂骨骼进行调整）


最后调整手腕和手指的骨骼

7.头部骨骼调整

调整脖子骨骼


调整头部的骨骼，大概在耳垂的位置


调整头部子骨骼在头发之上，垂直于头部骨骼


调整下颚骨，移动到耳朵中间的位置，再旋转Z轴，让其子骨骼在下巴的位置。


调整眼部骨骼，首先单独显示骨骼和眼球，并勾选显示眼球中心点。


再使用V健吸附，调整骨骼到中心：

8.设置骨骼数量，添加属性

  • 根骨骼

  • 腹部骨骼

  • 小腿骨骼（默认没有子骨骼，添加这个属性）

  • 脖子骨骼

  • 显示次级控制器（选择大腿，小腿，大臂，小臂）

9.创建控制器，蒙皮

创建控制器


蒙皮

点击这个工具选择所有的骨骼


然后加选身体模型，打开蒙皮选项面板


头部骨骼蒙皮（隐藏身体模型方便选择）


调整控制器形状大小

两种方式：

  • 选择点直接调整
  • 选择控制器，进行中心缩放（使用工具）


调整完后，使用镜像工具。

10.绘制权重

1.把所有权重给根骨骼

2.腹部骨骼

3.胸部骨骼

4.大腿骨骼（一层一层往下，小腿，脚踝，脚趾）


左脚刷完后，可以复制左边的权重到右边。



5.平滑胸部，腹部，大腿和大腿子骨骼权重



6.胸部和肩膀骨骼

然后是肩膀和大臂骨骼，一层层往下调整，直到手腕骨骼。


7.胸部和脖子骨骼


8.子骨骼绘制

根骨骼 - 跟子骨骼

腹部骨骼 - 腹部子骨骼

大臂 - 大臂子骨骼

大腿 - 大腿子骨骼

9.手部的骨骼绘制

从手腕骨骼开始：

手腕 - 大拇指

最后记得镜像到左边

10.衣服的蒙皮

先选择身体模型，运行mel命令：

select skinCluster -q -inf

然后加选衣服，进行蒙皮


然后先选择身体，在选择衣服，进行权重复制


裤子的操作和衣服一样。

11.对于头发的蒙皮

选择脖子骨骼和它所有子骨骼，然后加选头发



12.鞋子的权重复制

首先还是先选择身体模型，运行mel脚本，选择所有身体和骨骼。

然后加算两双鞋子，执行Bind Skin。

再选择脚的点，再选择鞋子的点，执行复制权重。

11.使用blendshape进行修形

比如肩部旋转，无法用权重很好的控制，就需要对它进行形状修正。


使用工具，把肩膀部分平滑


复制一份袖子的形态给左边，然后选择被复制出来的target，进行翻转


使用驱动关键帧

（注意不到选择到了IK，可以隐藏MotionSystem方便选择）



最后记得把驱动关键帧曲线打平


或者使用adv的修形骨骼进行修形

12.绑定后检查

  • 设置控制器颜色
  • 整理层级结构，并使其无法被选中
