将PyQt4代码转到PyQt5

最近在读这本经典的GUI编程书《Python Qt GUI快速编程》。

在这之前虽然已经写了很多的图形界面了，
但是一直没有系统的学过PyQt。
蹭着节假日期间，可以好好的咬咬这本书。

书中的代码示例是用的PyQt4，和即将淘汰的Python2。
而在学习的时候需要一点前瞻性，
需要考虑到今后对知识的使用，和它的时效。

SCP-079-TIP说：Python2已经停止维护了。
（而PyQt4也是会被淘汰的版本）

所以我们在看这本书的时候，就得把代码语法重写成Python3+PyQt5。

根据经验，一些常用的方法，可以直接从PyQt4转换到PyQt5.
比如在连接信号的时候：

# PyQt4
self.connect(self.lineedit,
             SIGNAL("returnPressed()"),
             self.updateUi)
# PyQt5
self.lineedit.returnPressed.connect(self.updateUi)

在发射信号的时候：

# PyQt4
self.emit(SIGNAL("acceptedList(QStringList)"), self.stringlist)

# PyQt5
self.acceptedList.emit(self.stringlist)

自定义信号的时候：

self.zeros = 0

# PyQt4
self.emit(SIGNAL("atzero"), self.zeros)

# PyQt5
atzero = pyqtSignal()
self.atzero.emit(self.zeros)

但是对于刚开始学习GUI编程的朋友，在进行重写转换的时候，
就会容易脑袋大，失去学习的动力。

为了解决重复的重写和给刚入门的新手指路，
这里在网上找到了一个可以直接进行版本转换的py代码。
http://www.vslcreations.com/2018/02/how-to-convert-pyqt4-code-to-pyqt5-code.html

下载后，可以输入下面的命令将PyQt4的代码转换为PyQt5：
python pyqt4topyqt5.py source.py -o output.py

这样就为我们省去了很多时间，在对照书里案例的时候，
可以不会为了这些琐事而消耗学习的专注度和兴趣。
