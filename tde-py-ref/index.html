3DEqualizer 命令参考

zhangly 2021-11-23 18:32:54
Categories： DCC > 3DEqualizer Tags：

窗口

# 创建一个自定义请求对象
req = tde4.createCustomRequester()

# 为req对象添加“文件浏览器”部件
tde4.addFileWidget(req, "filepath", "Browse", "*.obj")

# 为req对象添加"开关选项"部件
tde4.addToggleWidget(req, "faces_only", "Export Faces Only", 1)

# 布置req对象
tde4.postCustomRequester(req, "Export 3D Model as Obj…", 500, 0, "OK", "Cancel")

# 弹出一个问询选择对话框（1是标题，2是问题描述，3选择的按钮）
tde4.getQuestionRequester("1","2","3")

获取

# 获取当前激活摄像机id
tde4.getCurrentCamera()

# 获得当前的畸变模式
tde4.getLensLDModel(lens)

# 获取当前帧长度
tde4.getCameraNoFrame(cam)

# 获取当前Filmback w
tde4.getLensFBackWidth(lens)

# 获取当前Filmback h
tde4.getLensFBackHeight(lens)

# 获取指定帧的摄像机焦距
tde4.getCameraFocalLength(cam,1)

# 创建指定类别的摄像机
tde4.createCamera('SEQUENCE')

# 获取3DE软件路径
tde4.get3DEInstallPath()

# 获取当前指针帧数
tde4.getCurrentFrame(cam)

# 获取当前的点组
tde4.getCurrentPGroup()

# 获取指定点组已选择的模型列表
tde4.get3DModelList(pg,1)

# 获取当前点组类型
tde4.getPGroupType(pg)

# 获取选中的摄像机，如果没有任何选择，则返回列表
tde4.getCameraList()

设置

# 指定目标路径的素材
tde4.setCameraPath(cam, path)

# 设置素材的开始，结束帧数，步数
tde4.setCameraSequenceAttr(cam,st,end,step)

其它

# 打开指定文件
tde4.loadProject(r'/usr/share/home/*.3de')

# 清空控制台窗口
tde4.clearConsole()
